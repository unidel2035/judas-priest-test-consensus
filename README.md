# test-consensus

This repository contains a React test application.

## Project Structure

- `react-app/`: The main React application created with Create React App
  - Source code and configuration for the React application
  - Contains all the React components, styles, and assets

## Getting Started

1. Navigate to the `react-app` directory: `cd react-app`
2. Install dependencies: `npm install`
3. Start the application: `npm start`

The application will be available at `http://localhost:3000`.
