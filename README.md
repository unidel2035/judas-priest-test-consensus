# Vue.js Demo Application

A simple demonstration Vue.js application created as part of Issue #18.

## Features

- **Counter Component**: Interactive counter with increment, decrement, and reset functionality
- **Todo List**: Basic todo management with add, delete, and toggle completion features
- **Responsive Design**: Clean, modern UI with Vue.js styling

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the displayed URL (typically http://localhost:5173)

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
